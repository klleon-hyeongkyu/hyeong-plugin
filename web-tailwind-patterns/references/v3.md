# Tailwind CSS v3 패턴

## 설정 파일 (필수)

```javascript
// tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './src/**/*.{js,ts,jsx,tsx}',
    './pages/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          600: '#2563eb',
        },
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
  plugins: [],
};
```

## CSS 임포트

```css
/* globals.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

@layer base {
  :root {
    --background: #ffffff;
    --foreground: #000000;
  }
}
```

## 그라디언트

```tsx
// v3 문법
bg-gradient-to-r from-pink-400 to-rose-400
bg-gradient-to-br from-pink-400 via-red-400 to-orange-400
```

### 방향

| Class | 방향 |
|-------|------|
| `bg-gradient-to-t` | 위로 |
| `bg-gradient-to-tr` | 오른쪽 위 |
| `bg-gradient-to-r` | 오른쪽 |
| `bg-gradient-to-br` | 오른쪽 아래 |
| `bg-gradient-to-b` | 아래 |
| `bg-gradient-to-bl` | 왼쪽 아래 |
| `bg-gradient-to-l` | 왼쪽 |
| `bg-gradient-to-tl` | 왼쪽 위 |

## 커스텀 테마 확장

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      spacing: {
        '128': '32rem',
        '144': '36rem',
      },
      borderRadius: {
        '4xl': '2rem',
      },
      fontSize: {
        'xxs': '0.625rem',
      },
      animation: {
        'spin-slow': 'spin 3s linear infinite',
        'bounce-slow': 'bounce 2s infinite',
      },
      keyframes: {
        wiggle: {
          '0%, 100%': { transform: 'rotate(-3deg)' },
          '50%': { transform: 'rotate(3deg)' },
        },
      },
    },
  },
};
```

## 플러그인 사용

```javascript
// tailwind.config.js
const plugin = require('tailwindcss/plugin');

module.exports = {
  plugins: [
    require('@tailwindcss/forms'),
    require('@tailwindcss/typography'),
    require('@tailwindcss/aspect-ratio'),

    // 커스텀 플러그인
    plugin(function({ addUtilities }) {
      addUtilities({
        '.hide-scrollbar': {
          '-ms-overflow-style': 'none',
          'scrollbar-width': 'none',
          '&::-webkit-scrollbar': {
            display: 'none',
          },
        },
      });
    }),
  ],
};
```

## JIT 모드 (v3 기본값)

```javascript
// tailwind.config.js
module.exports = {
  mode: 'jit',  // v3에서는 기본값
  // Arbitrary values
};
```

### Arbitrary Values

```tsx
// 임의의 값 사용
<div className="w-[137px] h-[50px]" />
<div className="bg-[#1da1f2]" />
<div className="text-[22px]" />
<div className="p-[5px_10px_15px_20px]" />
<div className="grid-cols-[1fr_500px_2fr]" />
```

## 다크모드 설정

```javascript
// tailwind.config.js
module.exports = {
  darkMode: 'class',  // 또는 'media'
};
```

```tsx
// class 기반 사용
<html className="dark">
  <body className="bg-white dark:bg-gray-900">
    ...
  </body>
</html>
```

## @apply 지시자

```css
@layer components {
  .btn-primary {
    @apply px-4 py-2 bg-blue-500 text-white rounded-lg;
    @apply hover:bg-blue-600 focus:outline-none focus:ring-2;
    @apply transition-colors duration-200;
  }
}
```

---

## 주의사항

- `tailwind.config.js` 필수
- JIT 모드가 v3 기본값
- `@tailwind base/components/utilities` 임포트 문법 사용
- 그라디언트는 `bg-gradient-to-*` 문법
