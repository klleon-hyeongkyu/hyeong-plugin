# Tailwind CSS v4 패턴

## v3 → v4 주요 변경점

| 항목 | v3 (구) | v4 (신) |
|------|---------|---------|
| 테마 정의 | `tailwind.config.js` | `@theme` in CSS |
| 그라디언트 | `bg-gradient-to-r` | `bg-linear-to-r` |
| 설정 파일 | 필수 | 선택 (CSS 기반) |
| 임포트 | `@tailwind base/components/utilities` | `@import 'tailwindcss'` |
| 플러그인 | `plugins: []` in config | `@plugin` in CSS |

---

## CSS 임포트

```css
/* globals.css */
@import 'tailwindcss';
```

---

## @theme 정의

```css
/* globals.css */
@import 'tailwindcss';

@theme {
  --color-primary: #3579CC;
  --color-secondary: #FF6B6B;
  --color-background: #1e1e1e;
  --font-display: 'Inter', sans-serif;

  /* spacing */
  --spacing-128: 32rem;

  /* border-radius */
  --radius-4xl: 2rem;
}
```

### @theme inline (런타임 변수 참조)

```css
@theme inline {
  --color-background: var(--background);
  --color-foreground: var(--foreground);
}
```

### 사용

```tsx
<div className="bg-primary text-secondary" />
<div className="font-display" />
```

---

## 그라디언트

```tsx
// v4 문법
bg-linear-to-r from-pink-400 to-rose-400
bg-linear-to-br from-pink-400 via-red-400 to-orange-400
```

### 방향

| Class | 방향 |
|-------|------|
| `bg-linear-to-t` | 위로 |
| `bg-linear-to-tr` | 오른쪽 위 |
| `bg-linear-to-r` | 오른쪽 |
| `bg-linear-to-br` | 오른쪽 아래 |
| `bg-linear-to-b` | 아래 |
| `bg-linear-to-bl` | 왼쪽 아래 |
| `bg-linear-to-l` | 왼쪽 |
| `bg-linear-to-tl` | 왼쪽 위 |

### 커스텀 각도

```tsx
bg-linear-[45deg] from-blue-500 to-purple-500
```

## 플러그인 사용

```css
@import 'tailwindcss';

@plugin 'tailwind-scrollbar' {
  nocompatible: true;
  preferredstrategy: 'pseudoelements';
}

@plugin '@tailwindcss/typography';
```

---

## @layer utilities

```css
@layer utilities {
  .hide-scrollbar {
    -ms-overflow-style: none;
    scrollbar-width: none;
  }
  .hide-scrollbar::-webkit-scrollbar {
    display: none;
  }

  .text-balance {
    text-wrap: balance;
  }
}
```

---

## @layer components

```css
@layer components {
  .btn-primary {
    @apply px-4 py-2 bg-primary text-white rounded-lg;
    @apply hover:bg-primary/90 focus:outline-none focus:ring-2;
    @apply transition-colors duration-200;
  }
}
```

## 색상 투명도 (v4)

```tsx
// 슬래시 문법
bg-primary/50     // 50% opacity
text-white/80     // 80% opacity
border-gray-500/30

// 예시
<div className="bg-primary-40/90 text-white/80" />
```

## Container Queries (v4 기본 지원)

```tsx
// @container 사용
<div className="@container">
  <div className="@md:flex @lg:grid @lg:grid-cols-2">
    Content
  </div>
</div>
```

## 3D Transforms (v4)

```tsx
<div className="rotate-x-45 rotate-y-30 perspective-500" />
```

## 마이그레이션 체크리스트

1. ✅ `@tailwind` → `@import 'tailwindcss'`
2. ✅ `tailwind.config.js` → `@theme` in CSS
3. ✅ `bg-gradient-to-*` → `bg-linear-to-*`
4. ✅ `plugins: []` → `@plugin`
5. ✅ 색상 opacity: `bg-blue-500/50` 문법 확인

---

## 주의사항

- `tailwind.config.js`는 선택사항 (CSS 기반 설정 권장)
- 그라디언트는 `bg-linear-to-*` 문법 사용
- 런타임 CSS 변수 참조 시 `@theme inline` 사용
- v3 문법과 혼용 시 마이그레이션 도구 활용
