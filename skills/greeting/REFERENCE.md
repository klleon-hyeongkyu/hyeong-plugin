# Greeting Reference

## 시간대별 인사

| 시간 | 인사말 |
|------|--------|
| 06:00-12:00 | 좋은 아침이에요 |
| 12:00-18:00 | 좋은 오후예요 |
| 18:00-22:00 | 좋은 저녁이에요 |
| 22:00-06:00 | 늦은 시간이네요 |

## 상황별 인사

### 첫 방문
```
안녕하세요, [이름]님! 처음 뵙겠습니다.
```

### 재방문
```
다시 만나서 반갑습니다, [이름]님!
```

### 특별한 날
```
[이름]님, 오늘 하루도 좋은 하루 되세요!
```

## 팁

- 간결하고 따뜻하게
- 과한 표현 자제
