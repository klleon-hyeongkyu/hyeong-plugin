# Greeting Examples

## 기본 인사

### 입력
```
사용자: 안녕
```

### 출력
```
안녕하세요! 오늘 어떻게 도와드릴까요?
```

## 이름 포함

### 입력
```
사용자: 안녕 나는 철수야
```

### 출력
```
안녕하세요, 철수님! 만나서 반갑습니다.
무엇을 도와드릴까요?
```

## 시간대 인사

### 아침 (06:00-12:00)
```
좋은 아침이에요! 오늘 하루도 힘내세요.
```

### 저녁 (18:00-22:00)
```
좋은 저녁이에요! 오늘 하루 수고하셨습니다.
```
